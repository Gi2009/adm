Termos de Uso e Política de Privacidade

O Kanoa está comprometido com a proteção dos seus dados pessoais e a transparência no tratamento das informações coletadas. Esta política explica como coletamos, usamos, armazenamos e protegemos seus dados de acordo com a Lei Geral de Proteção de Dados (LGPD).

1. Aceitação

Ao criar uma conta e utilizar o aplicativo Kanoa, o usuário declara estar de acordo com os presentes Termos de Uso e com a Política de Privacidade.

2. Coleta e Uso de Dados

  • Coletamos informações como nome, e-mail, telefone, localização e preferências de reserva.
  • Os dados são usados exclusivamente para cadastro, agendamento de experiências culturais e processamento de pagamento.
  • Não vendemos, alugamos ou compartilhamos os dados sem consentimento.

3. Proteção de Dados

  • As informações são protegidas por criptografia e armazenadas em servidores seguros.
  • Dados de pagamento são processados por terceiros certificados (ex: via API de pagamento seguro).

4. Direitos do Usuário

Acesso, correção, exclusão de dados e revogação do consentimento podem ser solicitados a qualquer momento.

5. Uso de Conteúdo

Os usuários concordam que o conteúdo disponibilizado pela plataforma pode ser utilizado dentro do escopo do projeto Kanoa.

6. Atualizações

O Kanoa poderá alterar estes Termos para refletir mudanças legais ou melhorias da plataforma, com aviso aos usuários.
